# MOABOOK 프로젝트 입니다.

### 사용 기술

1. **FE(Front End)**
    - `React`
    - `JavaScript(ECMA6)`
    
2. **BE(Back End)**
    - Programming Language : `JAVA -ver 11`
    - FrameWork : `Springboot`
    - ORM : `JPA` `(hibernate)`
    
3. **Build tool**
    - `gradle`

1. 오픈소스 로그인 API 연동
2. JPA를 활용한 DB 설계
3. SPA (Single Page Application)
